# Face Fitness Tracker 

A Python-based face fitness tracker that detects and counts facial exercises
like jaw movement, eyebrow raise, and cheek puff using real-time webcam input.

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## Features
- Real-time face landmark detection
- Exercise-specific calibration
- Rep counting using stage-based logic
- On-screen UI with exercise name and counter

## How to Run
```bash
python fitness_tracker.py
